Chapitre 04

Le chapitre 4 traite du World Wide Web et d'Internet. Le World Wide Web est un système hypertexte public sur Internet qui permet de consulter des pages accessibles sur des sites. Les principes de base du Web incluent les URL (Uniform Resource Locator), le protocole HTTP (HyperText Transfer Protocol) et le langage de description de pages web HTML (HyperText Markup Language). Une page Web est un document multimédia stocké dans un fichier HTML ou .htm/.xhtml qui est interprété et affiché par un navigateur. Un site web est un regroupement de pages Web autour d'une thématique commune, reliées entre elles par des liens hypertextes. Le langage HTML est un langage de description de pages web construit avec des balises et est interprété dans un navigateur. Le fichier HTML doit comporter au minimum quatre balises: html, head, title et body. Les titres et les paragraphes sont également inclus dans le langage HTM
